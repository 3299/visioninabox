This is visioninabox's starting configuration page. If you see this, good job! - you've correctly installed and started visioninabox.

This configuration tool will walk you through several calibration procedures and questions before you start using it.

If at any time you have to restart the computer this is running on, don't worry - information from each step is saved after pressing 'Next'.

calibration

Let's first calculate some parameters unique to your camera.

Download and print out this calibration pattern. Tape it to something flat, like a piece of plywood, so that it isn't distorted.

Now, hold the calibration pattern at different angles and click 'Capture'. Try to take at least 10 captures at different angles.
